✨ 星光播放器 V0.4.5 发布公告

简介：重构播放控制架构，完善遍数播放和同步对齐功能。

新增功能：
1. 每首重复遍数播放：支持设置每首歌的重复播放次数（预设 1/7/21/49/108 遍），四种播放模式下行为各异。
2. 同步/非同步开关移至播放页面，与播放模式按钮对称分布，切换时即时生效。
3. 非同步模式下进度条可拖拽调整，同步模式下进度条锁定，视觉上通过粗细和亮度区分。
4. 同步开关默认开启。

播放控制优化：
1. 重构为单曲队列模式，彻底解决遍数控制和切歌闪烁问题。
2. 四种播放模式（单曲循环、单曲播放、列表循环、列表播放）遍数控制全部正确。
3. 一轮遍数播放结束后，UI 停留在 N/N，手动播放开启新一轮。
4. 同步开关对所有播放模式统一生效（包括单曲循环）。

界面优化：
1. 播放页面改为三段式布局（唱片区 + 进度区 + 控制区），比例更协调。
2. 控制按钮改为菱形布局，消除误触风险。
3. 唱片自适应屏幕大小，修复小屏显示不全问题。
4. 进度条时间和遍数字体加大，遍数显示与歌名样式统一。

稳定性改进：
1. 修复跨运行时状态共享问题（Android Service 与 UI 隔离）。
2. 完善诊断日志系统，日志持久化到本地存储，App 重启不丢失。
3. 关键业务路径全面埋点，便于问题排查。

说明：
1. 安装前，先卸载之前的版本。
2. APK 支持安卓、鸿蒙操作系统。
3. 苹果手机版本已同步更新。
